HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • You can divide the page into sections using the "section" element.

CSS

  • HTML elements can be thought of as being in a "box model," which can be modified in CSS.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The border is the lining between the margin and the padding; this can have the color, thickness, and style modified.
  • External stylesheets must be linked in the head of the HTML element using the "link" element.

Git

  • git status: shows which Git branch you are on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: moves to the listed branch-name (code must be committed before you can move)
  • git add -A: stage all files in the working branch to be committed
  • git commit -m "added code to HTML file": commits code to the repository with a message.
  • git pull origin main: this pulls the current branch to the base branch

JavaScript

  • JavaScript is a powerful tool used to add interactivity to a site.
  • A variable is a named container that allows us to store data in our code.
  • Set variables in JS using the following: "var name_of_variable = variable_value"
  • You can output a message to the web console using "console.log(insert_message_here);"
  • Control flow is the order in which a computer executes code in a script.
  • You can use conditional "if" and "else if" elements to manage control flow in JS.
  • "if" and "else if" determine if a code is falsy (0, -0, On, "", null, undefined, NaN, or a Boolean data type with the value of false) or truthy (any other value including the word "true") and determine if it needs to execute a provided action.
  • An array is a single variable that is used to hold a group of (typically related) data.
  • You can set arrays using the following: "var name_of_array = [arrayvalue1, arrayvalue2, arrayvalue3]"
